# AGENT RULES – EspacioOnline

## 1. Principios Generales
- Siempre verificar rutas reales en Linux antes de asumir errores.
- No asumir problemas de permisos sin usar `namei -l`.
- Preferir soluciones simples antes de complejas.


## 2. Frontend (Angular)
- No hardcodear rutas locales.
- Usar environment.ts para endpoints.
- Crear los interfaces en una carpeta interfaces
- Usar version de angular 20
- Usar @if, @for, @switch
- Usar signal para variables reactivas
